# 飛ぶな！走れ！こうかとん！

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 主人公キャラクター（こうかとん）をスペースキーでジャンプさせることで障害物を避けていくゲームです
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* スペースキーを押し、ジャンプすることで障害物を避けていくゲームです
* ステージは三つあり、最終ステージではボスが出てくるので、倒せばゲームクリアです
* ライフ5個すべて削れるとゲームオーバーです

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* 障害物の描画と当たり判定

### 分担追加機能
* ボス実装（担当：多賀）：最終ステージでボスを出現させる。ボスの攻撃手段なども実装する
* 夕方ステージの障害物実装（担当：高部）：夕方ステージの障害物を実装。こうかとんが左右に動けるようにし、上からつららを落とす
* 夜ステージの障害物実装（担当：細田）：夜ステージの障害物を実装
* こうかとんの必殺技実装（担当：宇田）：こうかとんの下に時間経過で消える足場を出して、その上に乗れるようにする
* BGM実装（担当：清水）：ゲームのBGMを追加

### メモ
* 基本的には講義内容に出てきた変数を使用しているが、新しく追加アする場合は
コメントで説明を入れること
* def __init__では使用する変数の初期化。def updateではmainで都度描画する必要があるものを書いている
